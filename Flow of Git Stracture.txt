FOur Stage of Git

1.Untracked
2.Unmodified
3.Modified
4.Staged

When file is untracked, you have add the file to "Staged"
Then "Commit" the file. If you want to edit the file then again unmodified later stage the file. 



Commands
--------------------------------

1.Checkout: Get back deleted file (git checkout filename). To get numbers of files at a time (git checkout -f).

2.touch: Create file use touch command (git touch filename).

3.Log: To check how many commit done till.(git log) to check only last commit (git log -p -1)

4.Commit: if you want to commit the code (git commit -m). Suppose if you want commit the code directly instead of adding staged then use this command(git commit -a -m).

5.switch: git switch name (git checkout branch name)

6.merge: (git merge branch name).
